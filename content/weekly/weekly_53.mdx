## 简讯

- Opera 又又有新浏览器了，[Opera One](https://www.opera.com/one)，目前还在 EA 阶段，旨在探索新的设计，取代现在的 Opera 浏览器（Deja Vu…
- Edge 浏览器会[把用户访问的网站发送到 Bing 上](https://www.theverge.com/2023/4/25/23697532/microsoft-edge-browser-url-leak-bing-privacy)。。
- 吴恩达和 OpenAI 合作推出的免费 Prompt Engineering 课程，主要是叫你如何写 Prompt。
- StackBlitz 的 [WebContainer 现已支持在 iOS 和 iPad OS 的 Safari 上运行](https://blog.stackblitz.com/posts/webcontainers-are-now-supported-on-safari/)了，补全了最后一块版图，这意味着你可以用 iPad 和 iPhone 的浏览器来运行 Node.js 项目了。
- Deno [发布 Deno KV](https://deno.com/blog/kv)，key-value 云数据库，支持部署在服务器、Web App 和 Edge 环境上。
- 第二天，Vercel 也发布了[自家的 Vercel Storage 三件套](https://vercel.com/blog/vercel-storage)，KV、Postgres 和 Blob ，总的来说就是整合底层资源了，有网友评价其价格感人。
- 前端框架 Qwik [发布 1.0 版本](https://www.builder.io/blog/qwik-v1)。
