## 简讯

- Linux 基金会旗下[新成立了一个 Agentic AI 基金会](https://aaif.io/)，Anthropic 向其[捐赠了 MCP](https://www.anthropic.com/news/donating-the-model-context-protocol-and-establishing-of-the-agentic-ai-foundation) ，OpenAI 则[捐赠了 AGENTS.md](https://openai.com/index/agentic-ai-foundation/) 。
- Meta 的策略正在[从开源模型转向可盈利的 AI 模型](https://www.bloomberg.com/news/articles/2025-12-10/inside-meta-s-pivot-from-open-source-to-money-making-ai-model)，计划推出名为 Avocado 的新模型。
- 美国开源模型有了新玩家，[Essential AI 发布了 Rnj-1 8B 模型](https://essential.ai/research/rnj-1)，在同级别模型里相比 Agentic 能力出众。
- Mistral 发布了[代码模型 Devstral2 和一个 CLI Agent Mistral Vibe CLI](https://mistral.ai/news/devstral-2-vibe-cli) 。
- Qwen 发布了 [Qwen3-Omni-Flash-2025-12-01](https://qwen.ai/blog?id=qwen3-omni-flash-20251201)， 一个新一代的本地多模态大模型。
- 智谱[发布 GLM 4.6V 模型](https://z.ai/blog/glm-4.6v)，支持原生的多模态 tool use 能力。还发布了一个开源的 Phone use Agent，[Open AutoGLM](https://github.com/zai-org/Open-AutoGLM)。
- OpenAI [发布了 GPT5.2](https://openai.com/index/introducing-gpt-5-2/)，常规更新，发布一天后差评如潮。
- 随后 Google [发布了 Deep Research Agen](https://blog.google/technology/developers/deep-research-agent-gemini-api/)t ，同时还有新的 Interactions API 。
- 安全研究员在研究 React 上周的 React2Shell 漏洞时，[发现了 RSC 另外两个严重漏洞](https://react.dev/blog/2025/12/11/denial-of-service-and-source-code-exposure-in-react-server-components)。。。
