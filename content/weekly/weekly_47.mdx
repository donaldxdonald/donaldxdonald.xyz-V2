## 简讯

- React 新版文档[正式上线](https://react.dev/)了。
- OpenAI GPT-4 模型[上线](https://openai.com/product/gpt-4)，性能更强，增加了图片支持等等，还能识图生成 html 代码（要失业了
- 百度也发布了其 AI 大语言模型——[文心一言](https://yiyan.baidu.com/welcome) （全部页面都是大模型三个字。。
- 微软表示其实咱们的 New Bing 就是[用着 GPT-4 模型](https://blogs.bing.com/search/march_2023/Confirmed-the-new-Bing-runs-on-OpenAI%E2%80%99s-GPT-4)哒。紧接着，[Microsoft 365 Copilot](https://news.microsoft.com/reinventing-productivity/) 降临了，Office 套件全面接入 AI，生产力拉满。
- 从 Atom Shell 到 Electron ，这个跨平台应用框架 Electron 已经[十年了](https://www.electronjs.org/blog/10-years-of-electron#not-goma)。
- TypeScript 5.0 [正式版发布](https://devblogs.microsoft.com/typescript/announcing-typescript-5-0/)了。
- Chrome 111 支持了[新的 View Transitions API](https://developer.chrome.com/blog/spa-view-transitions-land/)，可以在 SPA 单页应用中有更好的过渡动效。
