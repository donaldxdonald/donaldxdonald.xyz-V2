- [Complete rewrite of ESLint](https://github.com/eslint/eslint/discussions/16557)

  > ESLint 第十年，作者宣布要重写了。支持更多环境（Node，Deno，浏览器等等），支持更多语言（或许 JS 只是其中之一），新 API ，插件，基于 AST 的 autofixing，当然还有用 Rust 写… 作为“现代前端”人手必备的基建之一，在前阵子的锈化（Rustify）浪潮里，就已经出现了 [rslint](https://github.com/rslint/rslint) 和 [rome](https://github.com/rome/tools) 这样的工具，但也还没这么快能用。
