## 工具

- [mquickjs](https://github.com/bellard/mquickjs)
  
    传奇开发者 Fabrice Bellard 新的项目 MicroQuickJS，一个针对嵌入式系统的 JavaScript 引擎，能够在只有 10 kB 内存的情况下编译和运行 JavaScript 应用。Redis 作者在[评论区提到](https://news.ycombinator.com/item?id=46367224#46368706)，如果这个工具在 2010 年就诞生的话，Redis 的脚本语言就会是 JavaScript 而不是 Lua 。
    
- [AI SDK 6](https://vercel.com/blog/ai-sdk-6)
  
    Vercel 的 AI SDK 更新到了 V6，主要是带来了 Agent 的抽象层。
